## 🚀 **Démarrer Maintenant: 3 Options Faciles**

### **🟢 OPTION 1: Le Plus Rapide (2 min)**
```
1. Ouvrez https://mermaid.live
2. Collez ce code du fichier use-case-diagram-mermaid.md
3. Cliquez "Download" → PNG
4. Prêt! ✅
```

### **🟡 OPTION 2: Le Plus Flexible (5 min)**
```
1. Ouvrez https://app.diagrams.net
2. Créez un nouveau diagramme
3. Utilisez l'interface graphique pour dessiner
4. Exportez en PNG/SVG/PDF
5. Personnalisez les couleurs et l'organisation
```

### **🔵 OPTION 3: L'Intégration Complète (10 min)**
```
1. Ouvrez VS Code
2. Allez Extensions → Recherchez "Mermaid Preview"
3. Installez l'extension
4. Ouvrez vos fichiers .md
5. Activez Preview (Ctrl+Shift+V)
6. Modifiez le code directement et visualisez
```
